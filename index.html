Welcome to Boostan.

This is an LaTeX template which are frequently used in our documentations. Boostan is an documentation standard for book, letter, article, presentation and so on. This standard is designed to well-formed all documents and to separate the document writing from document beautifierring.

Her is the project assumptions:

  • To separate writing from design.
  • To support team work on documentation.
  • To increase the document portability.
  • To increase the document reusability.

Get the latst version

To get the latest version of Boostan run the following commands:

$ mkdir -p ~/git
$ cd ~/git
$ git clone https://github.com/phoenix-scholars/Boostan

For more information about installing see the https://github.com/phoenix-scholars/Boostan/wiki.
